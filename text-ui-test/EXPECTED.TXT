Welcome to the user guide. This guide has all the commands that are necessary to operate Duke

Main commands : 

Todo : adds a todo task to the task list :-> todo <description of the task> 
Deadline : adds a deadline task to the task list :-> deadline <description of the deadline> /by <time of the deadline> 
Event : adds an event task to the task list :-> event <description of the event> /at <time of the event> 

Other commands: 

Mark : marks a task as done :-> mark <task number> 
Unmark : marks a task as not done :-> unmark <task number> 
Delete : deletes a task :-> delete <task number>
Help : brings up this display :-> help
Bye : closes Duke :-> bye
